# ☕ PK Treats Cafe Website

A modern and responsive cafe website developed using **HTML, CSS, Bootstrap, and JavaScript**. The website provides an attractive user interface with a responsive layout, interactive menu filtering, and a seamless browsing experience across desktop, tablet, and mobile devices.

---

## 🌐 Live Demo

🔗 https://priyu-code14.github.io/PK-Treats-Cafe/

---

## 📌 Features

- 🏠 Responsive Home Page
- ☕ About Us Section
- 🍕 Interactive Menu with Category Filtering
- 🍰 Modern Food Cards
- 🛎️ Services Section
- 📞 Contact Page
- 📱 Fully Responsive Design
- 🎨 Attractive UI with Bootstrap Components
- ⚡ Smooth Navigation
- 📱 Mobile-Friendly Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons

## 🚀 How to Run

### Option 1

Download or clone this repository and open **index.html** in your browser.

### Option 2

Clone using Git:

```bash
git clone https://github.com/Priyu-code14/PK-Treats-Cafe.git
```

Open the project folder and launch **index.html** using your preferred browser or Live Server in Visual Studio Code.
